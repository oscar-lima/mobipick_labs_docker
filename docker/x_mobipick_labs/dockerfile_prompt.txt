- Prefer apt over apt-get
- your base images are:
  ozkrelo/[focal-snapshot (Ubuntu 20.04), noetic-ros-core (ROS1), mir, mobipick, mobipick_labs]

   "# This Dockerfile implements the provided natural language steps in
      dockerfile_prompt.txt,
      manual editing is not recommended."
- Base image to use: ozkrelo/mobipick_labs:noetic
- Output only one fenced code block labeled Dockerfile, with no extra text.

